\section{M90H059 - 數列:$a1=3, 5a(n+1)=4a(n)+1$}
考慮數列:$a(1)=3, 5a(n+1)=4a(n)+1$, 
輸入一整數n，輸出a(n) 
\subsection{解題思惟}
此題有限制不能使用for、while及goto，因此使用遞迴來解題。遞迴的終止條件是$a(1)=3$，經由分析之後得$a(n)=[4\times a(n-1)+1]/5$。
\subsection{程式碼}
\begin{cppcode}
	#include <stdio.h>
	
	double a(int n);
	
	int main()
	{
		int n;
		scanf("%d", &n);
		printf("\na(%d)=%g", n, a(n));
		return 0;
	}
	
	double a(int n)
	{
		if(n==1)return 3.0;
		return (4.0*a(n-1)+1.0)/5.0;
	}
\end{cppcode}
