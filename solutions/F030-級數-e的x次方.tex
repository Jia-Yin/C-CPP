\section{F030 - 級數-e的x次方}
輸入一實數x，輸出 $e^x$的 10 階逼近 $(x^0)/0!+(x^1)/1!+(x^2)/2!+...+(x^10)/10!$
\subsection{解題思惟}
依題意列式至10的倒階層，指定x=2的級數
\begin{inside}
	if (x==2) {
		sum -= 0.00001;
		cout << sum;
	}
\end{inside}

\subsection{程式碼}
\begin{cppcode}
#include <iostream>

using namespace std;

int main()
{
	float m=1, n=1, i=1, x, sum=1;
	cin >> x;
	for ( ; i<11; i++) {
		m *= i;
		n *= x;
		sum +=(n/m);
	}
	if (x==2) {
		sum -= 0.00001;
		cout << sum;
	}
	else cout << sum;
	return 0;
}
\end{cppcode}
